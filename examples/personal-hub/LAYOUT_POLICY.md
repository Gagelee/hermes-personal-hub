# Demo Layout Policy

Status: Public demo policy  
Scope: Agent-readable layout guidance for the sanitized demo Hub.

## Orientation

Use Life OS orientation with a stable home surface and adaptive content inside each semantic zone.

## Home Zones

- Now: recent project progress and the latest demo digest.
- Worlds: public project world, synthetic fitness world, and fictional finance review candidate.
- Operations: automation watch and agent growth log.

## Demo Rules

- Keep private-looking data out of demo files.
- Keep finance protected even when data is fictional.
- Mark workflow candidates explicitly.
- Keep Growth Log factual and public.
- Use visual primitives from `theme.css`, `tokens.css`, and `primitives.css`.

## Export Boundary

This demo is hand-authored from the private working Hub shape. It is not a raw export. Any future exporter must validate, sanitize, and report changes before public commit.
